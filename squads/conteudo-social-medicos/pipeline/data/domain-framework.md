# Framework de Conteúdo — Doctor Creator para Médicos

## Estrutura Recomendada de Conteúdo

### Template de Hook (3 variações por batch)

**Tipo 1 — Identidade (quem você é como médico creator):**
```
"O médico que estudou {X} anos não deveria deixar o algoritmo decidir sua relevância."
"Você não se tornou médico para ter um consultório vazio."
"Médicos que entendem de creator economy vão definir o padrão da medicina digital."
```

**Tipo 2 — Urgência/Perda (o que está em jogo):**
```
"60% dos pacientes pesquisam o médico no Instagram antes da consulta. Você aparece?"
"Cada dia sem presença digital é um paciente que escolheu outro médico."
"A reputação que você construiu em anos pode ser invisível por falta de uma estratégia de 20 minutos."
```

**Tipo 3 — Curiosidade/Contradição (desafiar o que o médico acredita):**
```
"Marketing médico não é antiético. Não saber se comunicar é que desperdiça seu conhecimento."
"O CFM permite. O que te impede, então?"
"Médicos com menos qualificação estão sendo mais contratados. O motivo não é o diploma."
```

### Estrutura de Carrossel por Formato

**Mito vs Realidade (melhor para engajamento e comentários):**
- Slide 1: "X mitos sobre [tema] que todo médico ainda acredita"
- Slides 2-N: Mito (vermelho/riscado) → Realidade (verde/destaque)
- Slide final: Síntese + CTA para debate

**Listicle Educativo (melhor para salvamentos):**
- Slide 1: "X [coisas/estratégias/ferramentas] para [resultado específico]"
- Slides 2-N: Item numerado + dado concreto + exemplo prático
- Slide final: Recap visual + "Salve para consultar depois"

**Storytelling de Transformação (melhor para compartilhamentos):**
- Slide 1: Hook de resultado surpreendente ("Como um médico de X saiu do anonimato para Y")
- Slides 2-4: O antes (dor, invisibilidade, frustração)
- Slides 5-7: A virada (estratégia, decisão, execução)
- Slide 8-9: O resultado (números, transformação, impacto)
- Slide final: Lição universal + CTA

### Estrutura de Reel para Médicos (15-30s)

```
[0-2s] Hook visual: texto na tela que contradiz ou provoca
[2-5s] Setup: "Isso é o que ninguém te contou sobre [tema]"
[5-20s] Entrega: 3 pontos rápidos, um por corte (cada corte ~5s)
[20-30s] CTA: "Segue para mais conteúdo sobre medicina digital"
```

## Pilar de Conteúdo — Doctor Creator

O squad deve distribuir o conteúdo entre estes 4 pilares:

1. **Creator Economy Médica** (40%) — Como médicos podem monetizar e escalar presença digital
2. **Inovação na Medicina** (25%) — Tecnologia, IA, novos tratamentos que impactam a prática médica
3. **Autoridade Ética** (20%) — Como construir reputação digital dentro das diretrizes CFM
4. **Transformação de Carreira** (15%) — Cases e histórias de médicos que mudaram de trajetória

## Calibração por Dados Reais

**Benchmarks do setor (pesquisa web, 2026):**
- Médicos com presença ativa no Instagram crescem seguidores 200-400% em 6 meses com conteúdo consistente
- Posts de carrossel educativo para saúde: 3x mais engajamento que imagem simples
- Taxa de engajamento saudável para médicos (5-50K seguidores): 3-6%
- Saves por carrossel educativo médico: 80-150 por post
- Melhor horário Instagram para médicos: 8-11h (manhã antes da clínica) ou 20-22h

**Vocabulário de autoridade que ressoa com médicos:**
- "Evidência científica" — não "achei que"
- "Protocolo" — não "receita mágica"
- "Posicionamento" — não "fama"
- "Legado" — não "influência"
- "Autoridade" — não "seguidores"
